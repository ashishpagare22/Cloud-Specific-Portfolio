# Cloud Specific Portfolio

Premium personal portfolio for **Ashish Pagare**, built as a dark, cloud-inspired React landing page with interactive DevOps storytelling.

## Stack

- React + Vite
- Tailwind CSS
- Framer Motion
- Lucide React

## Features

- Full-screen hero with animated cloud/network background
- Sticky glassmorphism navbar with smooth scrolling
- Scroll progress indicator
- Interactive cloud architecture section with highlighted service layers
- DevOps project cards with modal-based live previews
- Auto-typing terminal section
- Grouped skills grid with icon glow states
- About and contact sections with resume download
- Fully responsive dark UI
